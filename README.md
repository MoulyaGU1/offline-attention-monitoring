# Attention Mapping Tool

A comprehensive tool for tracking and analyzing user attention patterns through various interaction signals.

## Features

- Keyboard, mouse, and scroll tracking
- Application focus monitoring
- Idle time detection
- Voice assistant integration
- Real-time dashboard
- Session analysis and reporting

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python run_app.py`

## Usage

- Launch the tool and start tracking
- View real-time metrics on the dashboard
- Generate session reports

## Troubleshooting

- Ensure all dependencies are installed
- Check system permissions for input tracking
- Verify microphone access for voice features# offline-attention-monitoring
